가상화에 부는 새 바람

물리적인 한개의 자원을 분할해 효율적으로 사용하거나, 물리적으로 다른 다수의 자원을 논리적으로 통합하는 기술  

예) 한 대의 컴퓨터 시스템에서 여러 개의 운영체제를 함께 운용할 수 있도록 지원한는 기술  

자원을 이용하는 이들이 어떤 자원들이 실제로 사용되는지를 구체적으로 알 필요가 없게 만드는데 핵심
